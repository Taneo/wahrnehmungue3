## Getting Started

# The npm way

<pre>
<code>
git clone git@github.com:Taneo/wahrnehmungue3.git

cd wahrnehmungue3

npm install

npm start
</code>
</pre>
# The docker way

<pre>
<code>
git clone git@github.com:Taneo/wahrnehmungue3.git

cd wahrnehmungue3

docker-compose up
</code>
</pre>
